ofxCameraSaveLoad
=====================================

Introduction
------------
Simple openFrameworks addon that lets yo save and load all the parameters of any ofCamera or ofEasyCam object.

Licence
-------
Read the `license.md` file.

Installation
------------
Just copy into the `openFrameworks/addons/` folder.

Dependencies
------------
None

Compatibility
------------
